Flexbox

Flex items

Una volta che abbiamo creato un container flex e gli abbiamo assegnato tutte le proprietà che ci servono per il layout desiderato, possiamo agire sui figli, quindi sui flex items, tramite altre proprietà CSS dedicate.

order

Serve a controllare l'ordine in cui gli elementi flex vengono visualizzati all'interno di un contenitore flex. Questa proprietà è utile quando vogliamo modificare l'ordine degli elementi senza cambiare la loro posizione nel markup HTML.

Valori:

  • Numero intero: Puoi assegnare un valore numerico a order, dove un numero più basso indica una posizione precedente nell'ordine di visualizzazione.
  • Valore predefinito: Il valore predefinito è 0. Gli elementi con lo stesso valore di order vengono visualizzati nell'ordine in cui appaiono nel markup HTML.

flex-grow

Questa proprietà ci permette di far espandere gli elementi di un container flex per tutto lo spazio disponibile e di determinare quanto ogni singolo elemento debba crescere rispetto agli altri fratelli. (Default 0)

flex-shrink

Al contrario di grow, definisce quanto un elemento flex dovrebbe ridursi rispetto agli altri fratelli nello stesso contenitore. Più grande il valore, più l'elemento viene ridotto stringendo la finestra del browser. (Default 1)

flex-basis

Definisce la dimensione iniziale di un elemento flex prima che lo spazio disponibile venga distribuito secondo le proprietà flex-grow e flex-shrink. In altre parole, flex-basis determina la dimensione di un elemento lungo l'asse principale del contenitore flex. Di base il valore è auto, il che significa che la dimensione iniziale dell'elemento è basata sul suo contenuto o su altre proprietà come width o height.

align-self

Consente di specificare l'allineamento di un singolo elemento all'interno di un contenitore flex lungo l'asse trasversale. Questa proprietà sovrascrive il valore di align-items impostato sul contenitore, permettendo un controllo più fine sull'allineamento degli elementi.
