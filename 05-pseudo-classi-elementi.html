Pseudo classi ed elementi

Pseudo classi ed elementi sono parole chiave che possiamo aggiungere ai nostri selettori CSS per targettare gli elementi solo quando sono in un determinato stato, per esempio quando l'utente ci passa sopra col cursore o quando l'elemento è il primo figlio del genitore.

Pseudo classi

Utilizziamo le pseudo classi aggiungendo alla fine del selettore due punti e la parola chiave che ci interessa.

Alcune delle pseudo classi più comuni:

  • :hover -> Si applica quando l'utente passa il cursore sopra un elemento.
  • :active -> Si applica quando un elemento è attivamente selezionato (ad esempio, durante un clic).
  • :focus -> Si applica quando un elemento ha il focus, per esempio quando un campo di input è selezionato.
  • :first-child -> Si applica al primo figlio di un elemento genitore.
  • :last-child -> Si applica all'ultimo figlio di un elemento genitore.
  • :nth-child(n) -> Seleziona il n-esimo figlio di un elemento padre. n può essere un numero, una parola chiave come odd o even, o una formula.

Pseudo elementi

Gli pseudo-elementi in CSS sono utilizzati per stilizzare parti specifiche di un elemento, senza la necessità di aggiungere markup HTML. A differenza delle pseudo-classi, che si riferiscono a stati o posizioni di un elemento, gli pseudo-elementi si concentrano su parti interne di un elemento stesso.

Utilizziamo gli pseudo elementi aggiungendo alla fine del selettore una coppia di due punti e la parola chiave che ci interessa.

Alcuni degli pseudo elementi più comuni:

  • ::before -> Aggiunge contenuto prima del contenuto dell'elemento selezionato.
  • ::after -> Aggiunge contenuto dopo il contenuto dell'elemento selezionato.
  • ::first-line -> Si applica alla prima riga di un blocco di testo.
  • ::first-letter -> Si applica alla prima lettera di un blocco di testo.
  • ::selection -> Si applica al testo selezionato dall'utente.
